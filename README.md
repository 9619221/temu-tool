# TEMU 选品工具

> AI 驱动的跨境电商选品决策助手

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-green.svg)

## 功能特性

- 🧮 **利润计算器** - 快速计算产品利润率
- 📈 **热门品类** - 实时热门品类数据
- 🔍 **产品搜索** - 搜索竞品数据
- 💡 **选品建议** - AI 智能推荐
- ⚠️ **风险提示** - 运营风险提醒
- 🚚 **物流方案** - 物流成本计算
- 🤖 **AI 生图** - Nano Banana 2 AI 生成产品图

## 安装

### 方式一：直接运行（开发模式）

```bash
# 安装依赖
npm install

# 启动
npm start
```

### 方式二：打包安装

```bash
# 安装依赖
npm install

# 打包成 exe
npm run build
```

打包完成后在 `dist` 文件夹中找到 `TEMU选品工具-Setup.exe`

## 配置

首次运行需要配置 API Token：

1. 打开软件设置
2. 输入 Replicate API Token（用于 AI 生图功能）
3. 保存即可
